Descripción del Proyecto
Esta aplicación permite a los amantes del café explorar, compartir y personalizar recetas de café. Posee una navegación intuitiva, gestión de recetas personalizadas, y funciones de persistencia de datos para almacenar recetas, imágenes y todos los datos de la receta como tal. La aplicación facilita la mejora de habilidades en la preparación de café, ya que permite guardar, editar y consultar recetas, además de registrar el historial de preparaciones.

Funcionalidades Principales
Navegación y Diseño de Pantallas:

Interfaz organizada con jerarquía clara y accesible.
Pantalla de inicio que muestra las recetas recientes y opciones para gestionar las recetas propias.
Icono personalizado y paleta de colores específica para una identidad visual atractiva.
Gestión de Recetas:

Registro y edición de recetas(Esto no se logro implementar por el tiempo de manera funcional), incluyendo ingredientes, pasos, duración de preparación, e imágenes.
Guardado persistente de las recetas seleccionadas y las más preparadas.

Características de Usuario:

Integración de temporizadores y resumen de productos necesarios.
Opciones para agregar fotos a cada receta y revisarlas posteriormente.
Posibilidad de compartir recetas y listas de productos con otras aplicaciones.
Opinión y Retroalimentación de Usuarios:

Pantalla de “Tu Opinión” que permite a los usuarios dar feedback sobre la usabilidad, contenido y diseño.
Las respuestas de la encuesta se envían al correo del desarrollador para análisis.
Acceso a Información Externa
La aplicación permite a los usuarios compartir recetas y productos a través de otras aplicaciones de mensajería o redes sociales, para lo cual explora las opciones de compatibilidad de dispositivos y aplicaciones externas.

Instalación
Descargar el archivo APK desde el enlace provisto.
Permitir instalaciones de “orígenes desconocidos” en el dispositivo.
Ejecutar el archivo APK para instalar la aplicación.
