Requerimientos Funcionales:
La aplicacion posee un listado de recetas personalizables.
La aplicacion posee la capacidad de guardar recetas favoritas.
La aplicacion permite crear guias predefinidas y adaptables a los diferentes cafes.
La aplicacion permite al usuario modificar recetas segun prefiera.

Requerimientos No Funcionales:

Aplicacion para dispositivos mobiles.
Interfaz facil de usar.
Capacidad de maneja de una gran cantidad de recetas.
